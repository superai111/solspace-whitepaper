1. Overview

Solspace is an on-chain gaming protocol built on Solana, starting with a multiplier-based game inspired by the ‚Äúcrash‚Äù genre. The project explores a community-owned game model where players, builders, and early supporters participate in shaping and growing the ecosystem through the native SOLS token.

2. Vision

  • On-chain logic where possible
  • Transparent recording of in-game activity
  • Community-aligned incentives
  • Progressive decentralization

The long-term vision is a game protocol guided by its ecosystem rather than a single centralized entity.

3. Why Solana

  • High throughput suitable for real-time games
  • Low transaction costs
  • Rich wallet ecosystem (Phantom, Solflare, Backpack)
  • Strong tooling for on-chain and off-chain development

4. Solspace Game (Overview)

  • Each round starts at 1.00x
  • The multiplier increases over time
  • Players may exit at any moment
  • The round ends when the multiplier crashes

Initially, some game logic may operate off-chain for performance reasons, with a roadmap toward deeper on-chain verification.

5. Token SOLS

SOLS is the native token of the Solspace ecosystem, designed primarily for governance, coordination, and representing early participation.
